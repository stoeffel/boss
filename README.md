# Boss

![](https://cloud.githubusercontent.com/assets/56996/21707209/d5732d80-d410-11e6-82ff-17d2c32b20c1.gif)

## Install

With [fisherman]

```fish
fisher boss
```

## Features

* Git
    * Dirty
    * Staged
    * Unpushed commits (ahead)
    * Unpulled commits (behind)
    * Unpulled and unpushed commits (diverged)
    * Detached HEAD
    * Branch name
* $status
* $CMD_DURATION
* $status in context
* Background jobs
* Superpowers (sudo)
* Host information
